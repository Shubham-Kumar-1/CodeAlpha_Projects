# CodeAlpha Projects

A collection of three web applications built with modern JavaScript frameworks and technologies.

## Projects

### 1. AURASTORE
An e-commerce store application with client and server architecture.

**Tech Stack:**
- Frontend: Vanilla JavaScript, HTML5, CSS3
- Backend: Node.js
- Database: Database integration (see server/db.js)

**Structure:**
- `client/` - Frontend application with Vite setup
- `server/` - Backend API and database configuration

**Getting Started:**
```bash
cd AURASTORE/client
npm install
npm run dev

# In another terminal
cd AURASTORE/server
npm install
npm start
```

---

### 2. Nexus
A modern web application built with Vite and vanilla JavaScript.

**Tech Stack:**
- Frontend: Vite, Vanilla JavaScript, CSS3
- Server: Express.js with hot module reloading

**Features:**
- Fast development server with Vite
- Component-based architecture
- Asset management

**Getting Started:**
```bash
cd Nexus
npm install
npm run dev
```

---

### 3. NexusSocial
A social platform application with file upload capabilities.

**Tech Stack:**
- Frontend: Vanilla JavaScript, HTML5, CSS3
- Backend: Node.js with Express
- File Storage: Local uploads directory

**Features:**
- User interaction platform
- File upload functionality
- Social features

**Structure:**
- `public/` - Frontend files and assets
- `uploads/` - User-uploaded files storage
- `server.js` - Main application server

**Getting Started:**
```bash
cd NexusSocial
npm install
npm start
```
